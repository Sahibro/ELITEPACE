# ⚡ ElitePace — Premium Presentation Timer

## 🚀 Quick Start (3 Steps)

### Step 1: Firebase Setup (Free)
1. Go to console.firebase.google.com
2. Create new project (free)
3. Enable Realtime Database
4. Copy config to firebase-config.js

### Step 2: Deploy
**Netlify:** Drag & drop folder → Live!
**Vercel:** Push to GitHub → Auto deploy!
**GitHub Pages:** Upload files → Done!
**cPanel:** Upload via FTP → Done!

### Step 3: Use
1. Open website → Click "Create New Session"
2. Share QR code with Speaker
3. Speaker opens link → Timer appears
4. Manager controls timer → Speaker sees colors!

## ⌨️ Keyboard Shortcuts (Manager)
| Key | Action |
|-----|--------|
| `Space` | Start / Pause |
| `Ctrl + R` | Reset timer |
| `Ctrl + Enter` | Send message |
| `Escape` | Close modals |

## 🎨 Color Psychology
| Color | Meaning |
|-------|---------|
| ⚫ Black | Normal — presentation in progress |
| 🟡 Gold | Last 2 minutes — start wrapping up |
| 🔴 Red | Time up — stop presenting |

## 🛠️ Customization
- Warning threshold: Change `WARNING_THRESHOLD_SECONDS` in script.js
- Colors: Change CSS variables in style.css
- Branding: Replace "ElitePace" throughout
